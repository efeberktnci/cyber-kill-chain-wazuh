# Diagrams

Store editable sources and exported PNG/SVG files together. Every exported diagram must identify its corresponding source file and revision date.

## Planned Diagrams

- Logical SOC architecture
- VirtualBox network topology
- Telemetry and detection data flow
- End-to-end scenario workflow

## When This Directory Will Be Filled

This directory is intentionally staged instead of being populated with placeholder images.

- Baseline architecture and network documentation is already published under `docs/`
- scenario-specific telemetry-flow diagrams will begin once two or more core attack phases are complete
- final executive workflow and incident-timeline diagrams will be added during the cross-scenario reporting stage

This keeps the visual layer aligned with validated content instead of speculative design.
