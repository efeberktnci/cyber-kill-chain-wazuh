# Reconnaissance Variant 03 - EDR + SIEM

## Status

Planned.

## Objective

Re-run the same reconnaissance theme and document what additional context an endpoint detection and response platform can provide when compared with host-based firewall telemetry and network IDS telemetry.

## Planned Value

This variant is intended to emphasize endpoint-side visibility:

- process context
- network context tied to endpoint activity
- analyst workflow differences between EDR and SIEM-first investigation

## Planned Deliverables

- attack execution evidence
- EDR-side detection or telemetry evidence
- SIEM-side validation or enrichment evidence
- MITRE ATT&CK mapping
- comparison notes against Variants 01 and 02

## Planned Detection Focus

- endpoint-centric recon visibility
- process and connection context
- comparative investigation depth versus firewall-only and IDS-driven approaches
